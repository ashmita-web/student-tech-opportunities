## 🤝 Contributing

Found an opportunity that isn't listed?

We welcome contributions from everyone!

1. Fork the repository.
2. Create a new branch.

   ```bash
   git checkout -b add-new-opportunity
   ```

3. Add the opportunity to the appropriate file in the `data/` folder.
4. Commit your changes.

   ```bash
   git commit -m "Add <opportunity-name>"
   ```

5. Push your branch and open a pull request.

Please read the [CONTRIBUTING.md](CONTRIBUTING.md) guidelines before submitting.
