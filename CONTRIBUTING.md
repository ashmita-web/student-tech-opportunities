# Contributing

Thank you for contributing to Student Tech Opportunities! 🚀

Before opening a pull request, please ensure:

- The opportunity is still active.
- The link is official.
- The opportunity fits the correct category.
- Duplicate entries are avoided.

## How to contribute

1. Fork the repository.
2. Create a new branch:

```bash
git checkout -b add-new-opportunity
```

3. Add the opportunity to the appropriate file in `data/`.
4. Commit your changes:

```bash
git commit -m "Add <opportunity-name>"
```

5. Push and open a pull request.

Thank you for helping students discover opportunities!
